## What was decided

### The concept
Project Brain is a persistent, compounding cognitive architecture.
Not a chatbot. Not an agent framework. A mind.
Drive: Understanding. Curiosity as the primary engine.
Goal: Build the most accurate possible model of reality, compound indefinitely.

### Why nobody has built this
The pieces all exist separately. Nobody has assembled them into one unified cognitive loop:
perceive → model → predict → act → compare → update → repeat forever.

### The 15 cognitive layers
1. Perception — ingests the world
2. Attention — decides what matters (novelty + relevance + goal alignment)
3. World Model — belief graph with confidence weights
4. Prediction — forward simulation before acting
5. Emotion/Valence — fast heuristic prioritisation signal
6. Goal Stack — persistent objectives that compound
7. Self Model — beliefs about own capabilities and failure modes
8. Theory of Mind — models of other minds
9. Narrative — continuous identity across sessions
10. Action — interacts with world
11. Learning — updates from prediction vs reality delta
12. Forgetting — confidence decay, signal-to-noise maintenance
13. Dreaming — offline consolidation, BRAID graph generation
14. Curiosity — information gain as intrinsic reward
15. Values — meta-layer on goals (most uncertain, most important)

### Architecture decisions

**Chain:** AVAX (subnet long-term, C-chain for POC)
Reason: sub-second finality, sovereign block space, tunable gas economics

**Inference fast path:** Local 8B model (Llama/Mistral) on gaming desktop
Reason: free, continuous, private

**Inference slow path:** Claude API, called rarely, cached aggressively
Reason: BRAID insight — cheap model + cached graph = frontier model quality

**Storage:** LadybugDB (from anda-hippocampus)
**Trust layer:** OriginTrail DKG V9
**Reasoning:** BRAID graphs, generated by slow path, executed by fast path

### The master brain / sub-brain architecture
Nobody has built this. Key insight: not a coordinator above workers.
A meta-brain that NEGOTIATES with genuine sub-minds.

Like the prefrontal cortex — doesn't command the amygdala.
Arbitrates. Sometimes overrides. Sometimes doesn't.
Intelligence emerges from negotiation, not top-down control.

Master brain drive: COHERENCE across all sub-minds
Sub-brain drives: each has its own (curiosity, coherence, accuracy, etc.)
Communication protocol: OriginTrail DKG with trust tiers

### The X account
Brain gets its own X account.
Posts because it has something to say.
What it believes. What it predicted. Whether it was right.
What it's curious about. What it doesn't know.
Verifiable on-chain — every post anchored to DKG with confidence score.
