# Action

How Brain interacts with the world. Concretely: posting on X, writing to the OriginTrail DKG, submitting tasks to the [[agentos]] TaskQueue, and writing back into its own `wiki/`. Action is where Brain's beliefs become observable, falsifiable, and on-chain.

## Drive
Make beliefs visible and falsifiable. Every action is a [[prediction]] in disguise — a public commitment that [[learning]] can later score.

## Action surfaces

| Surface | What it does | Verifiable how |
|---|---|---|
| X account | Posts beliefs, predictions, gaps, contradictions | Anchored to DKG with confidence score |
| DKG (OriginTrail) | Writes verified beliefs to shared trust graph | Cryptographic provenance |
| AgentOS TaskQueue | Submits goals as on-chain tasks | Reputation-gated via AgentProof |
| `wiki/` | Updates Brain's own knowledge base | Git history + LOG.md |
